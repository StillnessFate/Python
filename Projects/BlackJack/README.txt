#블랙잭 게임#

-실행 파일 : blackjack.py
-파이썬 기반 (파이썬 이터프리터 3.3 이상이 요구됨)
-실행시 컴퓨터에 필수 라이브러리(Pillow)가 설치되어있지 않은 경우 먼저 설치를 시도함(특수한 환경으로 설치가 실패할 경우 수동 설치가 요구됨)
-경기 기록은 실행 파일과 동일한 폴더에 텍스트 파일 형태로 저장되며, 자동 갱신됨
-권장 실행 환경은 Windows이고, macOS또한 크로스 플랫폼 작업이 완료되었으나 파이썬 기본 GUI의 문제로 성능이 다소 하락
-처음 실행시 입력하는 ID는 이전 기록이 있을 경우 전적을 불러오고, 없을 경우 새로운 프로필을 만듬
-'Auto Play'와 'Auto Regame'모드로 자동 게임 가능

<게임 진행 순서>
1.배팅 (오른쪽 아래 버튼)
2.딜 (게임 시작)
3.HIT or STAY
4.경기 결과 표시 및 정보 저장
5.재경기 or 종료
